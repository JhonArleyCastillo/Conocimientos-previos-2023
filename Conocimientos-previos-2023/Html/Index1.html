Etiquetas en html

Definición

HTML 5 es un Lenguaje de Marcado de Hipertexto (HyperText Markup Language) que funciona con las bases del HTML de siempre, pero que cuenta con etiquetas adicionales que permiten crear sitios webs más modernos y más compatibles con las tecnologías digitales actuales.

Es importante aclarar que HTML no es un lenguaje de programación como tal, sino una manera de estructurar, interpretar y desplegar el contenido de un documento web mediante etiquetas y atributos que le indican al navegador qué es cada elemento (por ejemplo un subtítulo, un párrafo, una frase en negritas, una lista de bullets, etc).

      utf-8

      Son caracteres universales que incluye casi cualquier idioma humano, la pagina web podra ser gestionada para visualizarse en cualquier idioma.

      head

      Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

      title

      Titulo de la pagina web que aparece en la barra de titulo del navegador.

      body

      Define el area donde se colocara el contenido de la pagina.
      Tambien es un contenedor que alberga texto, imagenes, enlaces, multimedia, formularios entre otros elementos html.

      header

      Es el encabezado del documento.

      footer

      Es el pie de la pagina

      article

      Contiene el articulo, representa un contenido autocontenido en una especie de seccion pero repreenta el articulo.

      span

      Es un elemento generico en linea, permite modificar ciertos atributos con su sintaxis y etiquetas predeterminadas.

      link

      Sirve para establecer enlaces entre documentos o archivos.

      script

      Conecta un documento JavaScript al documento html

      a href

      Inserta enlances externos e internos en la pagina web

      nav

      Representa una seccion de la pagina cuya tarea es proporcionar enlaces de navegacion, ya sea dentro o fuera de la pagina.

    Etiquetas de texto

      p

      Parrafo, distribuye texto en parrafos.

      h1-h6

      Etiquetas que permiten generar titulos segun su tamaño entre mas grande el número mas pequeña la letra.

        q

        Secciona una parte del texto, coloca el texto en cursiva.

        b

        Coloca el texto en negrita.

        br

        Salto de linea

        code

        Permite mostrar una parte del codigo en el front end

      Etiquetas de Agrupación

        section

        Permite agrupar contenido logico y coherente.

        div

        La etiquetadiv se utiliza para crear una capa o división que agrupa varias etiquetas HTML. Es muy versátil y se emplea para organizar y dar estructura al contenido.

        p

        Parrafo, distribuye texto en parrafos.

        main

        La etiqueta mainse utiliza para englobar la parte principal de la página. Es útil para identificar la sección central del contenido.

        hr

        La etiqueta hr crea una línea horizontal que indica una separación temática en el texto.

        ol

        La etiqueta ol crea una lista ordenada con números.

        ul

      La etiqueta ul crea una lista donde el orden no importa

        li

        Define o declara que los siguientes elementos pertenecen a una lista.

      Etiquetas de multimedia

        video

        Permite añadir un video al documento

        audio

        Agrega un archivo de audio al documento.

        img

        Permite agregar imagenes al documento con el elemento src.

      Etiquetas de tablas

        table

        Es el contenedor de la tabla, dentro de esta se incluye las filas y columnas.

        th

        Se utiliza para las celdas de cabecera de la tabla. Estas celdas suelen contener títulos o descripciones para las columnas.

        tr

        Representa una fila de la tabla. Cada fila contiene una serie de celdas.

      td

      Define una celda de datos dentro de una fila. Aquí es donde se coloca la información real.

      Etiquetas de Formularios

        form

        Contiene controles interactivos que permiten al usuario enviar informacion a un servidor web.

        label

        Representa el título o etiqueta de un elemento de control en el formulario. Por ejemplo, puedes usarlo para describir un campo de entrada o un botón.

        input

        Esta etiqueta crea varios tipos de campos de entrada, como campos de texto, casillas de verificación, botones de radio, etc.

        buttom

        Crea un botón en el formulario que puede ser utilizado para enviar o restablecer el formulario.

        select y option

        Estas etiquetas se utilizan para crear listas desplegables (selectores) y sus opciones.
